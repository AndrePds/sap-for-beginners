# SAP for Beginners

Learn SAP data structures and modern data engineering architectures using
free tools and synthetic datasets.

This repository teaches:

- SAP core tables
- Data modeling
- Synthetic data generation
- Lakehouse architecture with Databricks

Everything is **100% free to run.**

## What you will learn

1. SAP core concepts
2. SAP transactional tables
3. SAP data relationships
4. Building a Medallion architecture

## Repository Structure

docs/ → learning material  
datasets/ → synthetic SAP datasets  
databricks/ → pipelines and notebooks

## Requirements

- Databricks Free Edition
- Python
- ShadowTraffic

## Quick Start

1. Generate data
2. Upload to Databricks
3. Run the Bronze pipeline
